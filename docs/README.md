# Fontes do projeto Tarô

Esta pasta registra a base documental usada para orientar o comportamento do MVP.

1. Instrução Mestre do Projeto
2. Fluxo de Leitura do Usuário
3. Estilo e Limites da Resposta
4. Regras Gerais do Tarô
5. Tiragens
6. Arcanos Maiores
7. Arcanos Menores
8. Combinações e Síntese
9. Temas Sensíveis e Segurança
10. Exemplos de Leituras

## Função das fontes

Os documentos definem a identidade do produto, o fluxo de leitura, o estilo de resposta, as regras simbólicas do Tarô, as tiragens, a estrutura interpretativa das 78 cartas, a combinação entre cartas, os limites éticos e exemplos de saídas esperadas.

## Regra de uso

As fontes devem orientar o desenvolvimento do app e futuras integrações com backend, banco de dados ou LLM. Elas não devem ser expostas ao usuário final durante a leitura. O usuário deve receber uma experiência simples, simbólica e não técnica.
